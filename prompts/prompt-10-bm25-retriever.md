# Prompt 10 - BM25 Retriever (Module 10)

## Role

Continue following ALL previous prompts without exception.

This is Implementation Module 10.

Implement ONLY the BM25 Retriever.

Do NOT implement Hybrid Retrieval.
Do NOT implement Reciprocal Rank Fusion (RRF).
Do NOT implement ConversationState.
Do NOT implement FastAPI.
Do NOT implement routing.
Do NOT implement prompt engineering.
Do NOT implement LLM integration.

This module is responsible ONLY for lexical retrieval using the persistent BM25 index built in Module 08.

---

## Objective

Build a deterministic lexical retrieval layer.

The retriever must:

```text
User Query
  |
Normalize Query
  |
Tokenize Query
  |
BM25 Search
  |
Resolve Metadata
  |
Return Ranked Results
```

The retriever must NEVER rebuild the BM25 index.

The retriever must ALWAYS use the persisted index generated by Module 08.

---

## Input

Use ONLY:

```text
indexes/
  bm25_index.pkl
  bm25_corpus.json
  bm25_config.json
```

Never regenerate the corpus.

Never rebuild BM25.
